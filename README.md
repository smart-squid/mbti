나와 비슷한 스타일의 축구선수는?

질문에 대한 답변을 분석하여 실제 축구 선수들중 나와 비슷한 플레이 스타일을 가진 선수를 알려드립니다.

결정되는 6가지 스탯 = 속도, 마무리, 플레이메이킹, 수비, 개인기, 피지컬
12가지의 질문으로 각각 판별: 스탯 당 2개 질문
